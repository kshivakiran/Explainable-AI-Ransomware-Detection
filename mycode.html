AI Engineering Architecture
1. Anomaly Detection

IsolationForest learns benign manifolds. Ransomware triggers high anomaly scores (>0.7) by deviating from normal system call sequences.

2. Balanced Classification

RandomForest (200 trees) with balanced class weights processes 31 features to eliminate bias and ensure 100% recall on threats.

3. LIME Interpretability

LIME fits local surrogates around specific predictions, providing human-readable justification for every security alert generated.

4. Production Ready

Optimized sklearn pkl pipeline (~3MB) allows for high-performance deployment on cloud platforms with near-zero latency.
